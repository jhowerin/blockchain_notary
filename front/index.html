Welcome to the Blockchain Star Registry!

This program creates a Star Registry that allows users to enter a star on this blockchain. Users must have a wallet address and be able to sign/verify messages. Users must also know a few details regarding the star, including RA and DEC. Please see the Online Star Register to learn more.
